This service allows users to search for images by passing the search queries as parameter to the url

Example usage:

Send a parameter with the search query, in the following format :

https://image-abst.herokuapp.com/search/?q=elon+musk

To paginate through the results, pass an additional offset parameter :

https://image-abst.herokuapp.com/search/?q=elon+musk&offset=3

To view the 5 most recent search queries :

https://image-abst.herokuapp.com/recent
